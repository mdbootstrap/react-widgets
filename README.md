
Responsive React Widgets & Addons built with the latest Bootstrap 5. Calendar, eCommerce gallery, Chat, File upload, Countdown, WYSIWYG, Drag & drop, Filters, & more

Check out [React Bootstrap Widget Documentation](https://mdbootstrap.com/docs/react/extended/widgets/) for detailed instructions & even more examples.



[React Bootstrap widgets:
![React Bootstrap widgets](./basic.png)](https://mdbootstrap.com/docs/react/extended/widgets/)

## How to use?

1. Download MDB React - free UI KIT

2. Choose your favourite customized component and click on the image

3. Copy & paste the code into your MDB project

[▶️ Subscribe to YouTube channel for web development tutorials & resources](https://www.youtube.com/MDBootstrap?sub_confirmation=1)
